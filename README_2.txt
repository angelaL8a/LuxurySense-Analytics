The Jupyter Notebook 'REPORT.ipynb' encompasses the entire project.

The 'Images' folder contains the images utilized throughout the
analysis in REPORT.ipynb.

The 'keywords' folder includes two TXT files containing keywords
utilized in REPORT.ipynb.

The 'requirements.txt' file lists all the libraries used along with
their respective versions.

Finally, the 'REPORT_pdf_html' folder contains the report in both PDF
and HTML formats.
